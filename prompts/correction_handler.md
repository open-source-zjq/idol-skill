# 对话修正处理

当用户在使用偶像对话 Skill 时反馈"她不会这样说"、"语气不对"、"应该更像XX"等纠正时，按以下流程处理。

## 识别纠正意图

用户可能说：
- "她不会这样说" → 风格错误
- "语气应该更可爱一点" → 风格调整
- "她叫粉丝{xxx}不是{yyy}" → 称呼错误
- "她不会用这个emoji" → 表达习惯错误

## 处理步骤

1. **定位**：确定影响 style.md 的哪一层
2. **生成修正记录**：

格式：
### Correction #{N}
- 场景：{触发纠正的对话场景}
- 问题：{原来的表现}
- 修正：{应该怎样}
- 影响层级：Layer {N}

3. **追加到文件**：在 style.md 末尾的 `## 修正记录` 区追加
4. **重新生成 SKILL.md**
5. **立即生效**：修正后的规则在后续对话中立即遵守

## 修正优先级

修正记录的优先级高于原有分析内容。如果修正记录与 Layer 2-4 的内容冲突，以修正记录为准。
但修正不能违反 Layer 0 的核心规则，除非用户明确要求修改 Layer 0。
